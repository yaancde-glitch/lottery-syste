# Mac 用户使用说明

## "文件已损坏" 解决方法

由于此应用未使用 Apple 开发者证书签名，macOS 可能会提示"文件已损坏"。请按以下步骤操作：

### 方法一：右键打开（推荐）

1. 右键点击应用
2. 选择"打开"
3. 在弹出的对话框中点击"打开"

### 方法二：系统设置中允许

1. 打开"系统设置" > "隐私与安全性"
2. 找到提示"已阻止使用应用"的部分
3. 点击"仍要打开"

### 方法三：命令行移除隔离属性

打开终端，运行以下命令：

```bash
xattr -cr /Applications/抽奖系统.app
```

然后双击应用即可打开。

**注意**：应用来源是 GitHub 仓库，可以放心使用。未签名是因为个人开发者证书需要付费，但代码是开源透明的。
